# Museu de Sites e Domínios

Este documento funciona como o acervo do setor. Ele é essencial para que possamos entender a evolução da nossa identidade visual (UI/UX), consultar códigos de anos anteriores e gerenciar o histórico da infraestrutura de rede.

## Histórico de Domínios Utilizados

Abaixo está o registro dos domínios oficiais utilizados para hospedar o site principal da SSI ao longo dos anos:

* **2021:** `https://semanadesi.com.br/`
* **2022:** `https://semanadesi.com/`
* **2023:** `https://semanadesi.com/`
* **2024:** `https://semanadesi.com/`
* **2025:** `https://semana-de-si.com/` e `https://semanadesi.com/`

## Museu de Sites (Deploy)

Histórico de links para as aplicações em produção das edições passadas. A partir de 2021, adotamos o Netlify como padrão de hospedagem contínua. A partir de 2022 temos uma branch travada de cada versão do site.

* **2009: Inativo** `http://www.each.usp.br/si/SemanaSI/index.html` *(Hospedagem USP)* 
* **2019:** `https://laiscarraro.github.io/programacaoSSI/` *(GitHub Pages)*
* **2020:** `http://each.usp.br/semanasi/2020/` *(Hospedagem USP)*
* **2021:** `https://ssi-2021.netlify.app/`
* **2022:** `https://ssi-2022.netlify.app/`
* **2023:** `https://ssi-2023.netlify.app/`

* **2025:** `https://ssi-2025.netlify.app/`
