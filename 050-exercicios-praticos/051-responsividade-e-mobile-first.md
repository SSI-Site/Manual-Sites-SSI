# Exercício 01: Responsividade e Mobile First

Este exercício é o seu primeiro desafio prático no setor. O objetivo é aplicar os conceitos de HTML semântico, CSS Flexbox e, principalmente, a metodologia Mobile First que utilizamos em todos os projetos da SSI.

## O Desafio
Você deve criar uma página web simples e responsiva utilizando **somente HTML e CSS puros** (sem frameworks como Bootstrap ou bibliotecas como React por enquanto).

### Requisitos Mínimos:
1. **Estrutura de Cards:** A página deve conter, no mínimo, 3 cards informativos.
2. **Composição do Card:** Cada card deve possuir, pelo menos, uma imagem e um parágrafo de texto.
3. **Layout Flexbox:** Use obrigatoriamente o sistema de Flexbox para o alinhamento e distribuição dos elementos.
4. **Foco em Mobile First:** O código deve ser escrito pensando primeiro na tela do celular.
5. **Media Queries:** É fundamental que sejam usadas apenas condições baseadas no tamanho mínimo (`min-width`) para a definição das media queries. Isso garante que o estilo base seja o mobile e o desktop seja a "exceção" para telas maiores.

## Orientações Adicionais
* Busque imagens leves e que façam sentido entre si.
* Tente replicar o comportamento de um site real: uma seção de cabeçalho, a seção de cards no corpo e um rodapé simples.
* Use o "Inspecionar Elemento" do navegador para testar como sua página se comporta em diferentes larguras de tela (iPhone, Pixel, iPad e Desktop).
