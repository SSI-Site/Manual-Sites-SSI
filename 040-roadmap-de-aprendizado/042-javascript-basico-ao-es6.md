# Nível 2: JavaScript Básico ao ES6+

**⚠️ Importante:** Este roadmap não é um curso completo. Ele serve como um guia para direcionar seus estudos nos pontos que realmente utilizamos no dia a dia do desenvolvimento do Site-SSI e da Dashboard. O JavaScript é a linguagem que dita o comportamento de tudo o que fazemos, por isso, dedique um tempo extra aqui.

## 1. JavaScript Básico e Conceitos Fundamentais
Antes de mergulhar em frameworks, você precisa entender como a linguagem funciona no navegador.

**Conceitos essenciais e Materiais de Estudo:**
* **O que é JavaScript e como funciona nos navegadores:** 
* **Variáveis e Constantes:** Entender como armazenar dados.
* **Tipagem Dinâmica e Tipos Primitivos:**
* **Arrays e Funções:** A base para manipular listas de dados e organizar o código.

    [Vídeo Sugerido contemplando todos os tópicos acima](https://www.youtube.com/watch?v=W6NZfCO5SIk)

* **Diferenças entre `var`, `let` e `const`:** [Artigo sugerido](https://www.freecodecamp.org/portuguese/news/var-let-e-const-qual-e-a-diferenca/)
* **Hoisting:** Como o JavaScript processa as declarações. [Artigo sugerido](https://medium.com/devzera/o-que-%C3%A9-hoisting-em-javascript-b8c629178518)


## 2. DOM (Document Object Model)
O DOM é a interface que permite ao JavaScript "conversar" com o HTML e o CSS. No React não manipulamos o DOM diretamente com frequência, mas entender o conceito é vital.

* **Manipulação do DOM:** [Artigo sugerido](https://tableless.com.br/entendendo-o-dom-document-object-model/) && [Vídeo sugerido](https://www.youtube.com/watch?v=y17RuWkWdn8)
* Extra: **Masterclass de Manipulação:** [Rocketseat - Masterclass](https://www.youtube.com/watch?v=UftSB4DaRU4)

## 3. JavaScript Assíncrono
O Site-SSI precisa buscar dados na API Saphira e no Firebase sem travar a tela do usuário. Para isso, usamos programação assíncrona.

**Conceitos essenciais e Materiais de Estudo:**
* **Promises e Async/Await:** Como lidar com operações que levam tempo (como chamadas de rede).
* **Fetching Data:** Como buscar dados de uma API.
* [Vídeo Sugerido contemplando todos os tópicos assíncronos](https://www.youtube.com/watch?v=670f71LTWpM)

## 4. Funcionalidades do ES6+ (Essencial para React)
O React utiliza extensivamente as atualizações modernas do JavaScript. Se você dominar estes pontos, o código do nosso projeto fará muito mais sentido.

**Conceitos essenciais e Materiais de Estudo:**
* **Guia Completo do ES6:** [Artigo sugerido](https://medium.com/@matheusml/o-guia-do-es6-tudo-que-voc%C3%AA-precisa-saber-8c287876325f)

* **React e ES6+:** [Documentação do Babel](https://babeljs.io/blog/2015/07/07/react-on-es6-plus)
