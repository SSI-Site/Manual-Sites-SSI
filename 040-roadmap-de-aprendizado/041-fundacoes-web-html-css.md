# Nível 1: Fundações Web (HTML e CSS)

**⚠️ Importante:** Este roadmap não é um curso completo ou um tutorial definitivo. A intenção deste documento é servir como uma fonte de consulta e orientação. O objetivo é dar um "norte" para que os membros do setor criem ou fortaleçam a base de conhecimento necessária para contribuir com o desenvolvimento do site. Sinta-se livre para estudar os conceitos por outras fontes e ir além do que está listado aqui!

Todo grande projeto web começa pelo básico. Mesmo que você já vá trabalhar com React e Next.js mais para frente, os navegadores só entendem a base da web. Se você pular esta etapa, terá muita dificuldade para debugar problemas no futuro.

## 2. CSS (Cascading Style Sheets)
O CSS é a "pele" e a "roupa" do nosso site. É o que transforma um texto simples em uma interface moderna.

**Conceitos essenciais para dominar:**
* O que são Cascading Style Sheets?
* Sintaxe do CSS
* Seletores
* Fontes
* Margin, padding & border (box-model)
* Position
* Alinhamento

📚 **Material de Estudo (CSS Base):** [Vídeo Sugerido contemplando todos os tópicos acima](https://www.youtube.com/watch?v=r11FflkQqJs)
